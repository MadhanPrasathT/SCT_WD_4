# To-Do List Web Application

## Project Overview
This is a simple and interactive **To-Do List Web Application** that helps users manage their daily tasks efficiently. Users can add tasks, set date and time, mark tasks as completed, edit tasks, and delete tasks as needed.

The application is designed to be user-friendly and is built using basic web technologies.

---

## Features
- Add new tasks
- Set date and time for tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Organize tasks in a list
- Simple and clean user interface

---

## Technologies Used
- HTML
- CSS
- JavaScript
